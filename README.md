# docker-apache-mysql-php7-laravel
Dockerized LAMP wrapper for Laravel projects based on PHP 7.4

## Features
* PHP 7.4
* MySQL 8
* Apache 2.4
* Cron config for Laravel apps
* `.htaccess` rewrite rules enabled

## Start (`start.bat`)
```sh
docker compose up -d --build
```

## Stop (`stop.bat`)
```sh
docker compsoe down
```
